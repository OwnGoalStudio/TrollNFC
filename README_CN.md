## 📱 使用条件

- **支持 NFC 的 iPhone**
- **iOS 14.0 及以上版本**
- **需要安装 TrollStore**

## 核心功能

- **读卡、写卡、模拟卡：** 一键搞定 NFC 卡片的所有操作
- **批量操作：** 支持同时处理多张卡片，效率翻倍
- **高级编辑：** 轻松查看和修改 NDEF 数据，包括 URL 记录
- **备份恢复：** 卡片数据一键导出，换机无忧
